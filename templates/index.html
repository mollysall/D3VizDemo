Fishbone Diagrams

This page was created by an intern at GCT.

This fishbone diagram can be used to visualize and sort chronological data based on category, location and date.

This visualization uses JavaScript, JQuery, Python, Flask, and the D3 Javascript library to create an interactive timeline that makes it easy to view only pertinent information.
